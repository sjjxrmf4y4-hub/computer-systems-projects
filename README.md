# Computer Systems Projects

Projects and notes focused on computer architecture, low-level programming, processor design, and memory systems.

## Topics

### MIPS Assembly Programming
Programs involving loops, functions, arrays, arithmetic, and memory access.

### CPU Pipelining
Instruction timing, hazards, forwarding, stalls, and execution stages.

### Cache Memory Systems
Hit/miss analysis, replacement policies, and performance concepts.

### Virtual Memory
TLB, page tables, address translation, and memory hierarchy.

### Computer Security Foundations
Stack smashing attacks, memory safety, and defensive concepts.

## Purpose

Built through coursework in Computer Organization and independent study.
