# Cache Examples

## Cache Concepts

Cache stores frequently used data closer to CPU for faster access.

## Terms

- Hit: Data found in cache
- Miss: Data not found in cache
- Block: Unit transferred from memory
- Set: Group of cache lines

## Example

Memory address accessed twice in short time often results in hit on second access.

## Policies

- Direct Mapped
- Set Associative
- Fully Associative

## Goal

Reduce average memory access time.
