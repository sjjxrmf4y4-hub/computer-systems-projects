# MIPS Base Converter Project

MIPS assembly project completed in Computer Organization using the QTSpim simulator.

## Project Overview

Developed multiple assembly programs culminating in a base converter that accepts unsigned decimal input and outputs:

- Binary representation
- Hexadecimal representation
- Repeated conversions through user-controlled looping

## Included Files

- `Program_1.asm` — Time-of-day classifier
- `Program_2.asm` — Sequence summation
- `Program_3.asm` — Decimal to hexadecimal digit converter
- `Base_converter.asm` — Decimal to binary and hexadecimal converter

## Technical Skills Demonstrated

- MIPS registers
- Branching and jump logic
- Loops
- Arithmetic instructions
- Memory addressing
- Lookup tables
- Syscalls for input/output
- Debugging in QTSpim

## Tools Used

- MIPS Assembly
- QTSpim

## Outcome

Strengthened understanding of low-level computation, processor operations, and assembly programming fundamentals.
