# MIPS Base Converter Project

MIPS assembly project completed in Computer Organization (CDA3100) using the QTSpim simulator.

## Project Overview

Developed multiple assembly programs culminating in a base converter that accepts unsigned decimal input and outputs:

- Binary representation
- Hexadecimal representation
- Repeated conversions through user-controlled looping

## Implemented Components

### Program 1: Time of Day Classifier
Determines whether an input hour represents:

- Morning
- Noon
- Afternoon
- Midnight

### Program 2: Sequence Summation
Calculates the sum of a numeric sequence using loop logic.

Example:
13 + 14 + 15 + 16

### Program 3: Decimal to Hex Digit
Uses a lookup table to convert values 0–15 into hexadecimal characters.

### Final Program: Base Converter

Converts decimal integers into:

- Binary
- Hexadecimal (`0x` format)

Allows multiple conversions in a single execution.

## Technical Skills Demonstrated

- MIPS registers
- Branching and jump logic
- Loops
- Arithmetic instructions
- Memory addressing
- Lookup tables
- Syscalls for input/output
- Debugging in QTSpim

## Tools Used

- MIPS Assembly Language
- QTSpim

Strengthened understanding of low-level computation, processor operations, and assembly programming fundamentals.
